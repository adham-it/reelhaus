REELHAUS – Reels für die "Arbeiten"-Sektion
============================================

Lege deine Videoclips einfach hier in diesen Ordner. Erwartete Dateinamen:

    reel-1.mp4   (optional: reel-1.jpg  = Standbild/Poster)
    reel-2.mp4   (optional: reel-2.jpg)
    reel-3.mp4   (optional: reel-3.jpg)
    reel-4.mp4   (optional: reel-4.jpg)
    reel-5.mp4   (optional: reel-5.jpg)
    reel-6.mp4   (optional: reel-6.jpg)

Sobald eine Datei vorhanden ist, wird sie automatisch im Handy-Mockup
abgespielt (stummgeschaltet, Loop, nur das gerade sichtbare Reel spielt).
Fehlt eine Datei, bleibt der farbige Verlauf als Platzhalter – nichts bricht.

Empfehlung:
- Format: MP4 (H.264 / AAC)
- Hochformat 9:16 (z. B. 1080 x 1920) sieht am besten aus
- Pro Clip < ~10 MB für flüssiges Laden über WLAN

Namen, Likes, Kommentare, Views & Caption pro Reel werden in
assets/main.js (WORK_POSTS) gesetzt – sag mir die echten Werte,
dann trage ich sie ein.
